
## Animation Guidelines

### Duration
- **Fast**: 200ms - UI feedback, toggles
- **Normal**: 300ms - Screen transitions
- **Slow**: 500ms - Complex animations

### Curves
- **easeIn**: Fade in animations
- **easeOut**: Fade out animations
- **easeInOutCubic**: Page transitions
- **spring**: Playful interactions

---

## Accessibility

### Touch Targets
- Minimum: 44x44 pixels
- Buttons: 48x48 pixels recommended
- Icon buttons: 40x40 minimum

### Contrast Ratios
- Text/Background: 4.5:1 minimum
- Large Text/Background: 3:1 minimum
- Icon/Background: 3:1 minimum

### Font Sizes
- Minimum body text: 14px
- Minimum labels: 12px
